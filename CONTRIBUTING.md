# Contributing to KartSan

Thanks for your interest in making KartSan better! Here's how you can help.

## Ways to contribute

### Report issues
If KartSan gives incorrect advice, misses something, or could be better:
- Open a GitHub issue
- Include what you asked, what KartSan said, and what would have been better
- If it's a Rotax rules issue, include the specific regulation reference

### Improve the knowledge base
The reference files in `skill/references/` contain KartSan's domain knowledge. If you spot:
- Outdated Rotax regulations (they change yearly)
- Missing tracks in the European track database
- Incorrect setup advice for specific chassis brands
- Better jetting/pressure guidelines based on your experience

...submit a PR with the correction and a brief explanation of why.

### Add track data
We especially welcome contributions of track knowledge:
- Corner-by-corner breakdowns of karting circuits
- GPS-derived track layouts
- Setup notes for specific venues

### Translate
KartSan currently supports English and Portuguese. Want to add your language?
- Add terminology translations to `skill/references/kid-communication.md`
- Add language detection cues to `skill/SKILL.md`
- Test with example prompts in your language

## Guidelines

- Keep advice safety-first, especially for young drivers
- All setup suggestions must comply with Rotax regulations
- Test your changes with the example prompts in `examples/`
- When updating regulations, cite the official source

## Code of conduct

Be kind. This project exists to help kids enjoy racing. Keep that spirit in everything you contribute.
